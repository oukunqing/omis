oTreeTable v1.1
树形表格

使用方法：

表格第一行如果是表头，请将单元格 设为 TH，而不是TD，第一行的TH不参与树形重建

1.1 增加了JSON格式容错处理，且lang属性可以加在 tr中，也可以加在tr行中第1个或第2个td中，对于没有加lang或lang错误的表格行原样复制并显示在前面。
    修改了保留表头的逻辑，可以保留多重表头（之前只能保留第一行的表头）

1.2 增加了指定单元格的功能

1.3 修复单元格合并列时出现的单元格不完整的问题，增加 是否单击+号-号图标的功能 clickIcon=true|false

1.4 增加了按层级展开方法 expandLevel(level)，使用方法详见示例

1.5 增加了页面级Cookie缓存功能（以防止刷新），默认启用缓存功能，如不需要缓存，设置第三个参数{isCache:false}
